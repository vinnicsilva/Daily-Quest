# Daily-Quest
Faça suas tarefas (ou quests) de uma forma divertida e engajante com esse To Do List gamificado!

=====================================================================================

OBJETIVO:

A aplicação tem como objetivo usar banco de dados integrados com MySQL e PDO em PHP para editar, excluir e criar tarefas/subtarefas e além de claro tratar cadastro de usuários.

=====================================================================================

LINGUAGENS:

- HTML
- CSS
- JS
- PHP
